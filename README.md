# Crawl Coursera

**Step 1: Clone repo**
```
git clone https://github.com/Joycean0301/Courseraaa.git
```

**Step 2: Upload extension**

Google Chrome -> Setting -> Extensions -> Manage Extensions -> Turn on **Developer mode** -> **Load unpacked**: Upload folder git

![Screenshot from 2023-06-02 03-03-15](https://github.com/Joycean0301/Courseraaa/assets/103662477/41333829-8eb7-487d-8411-23c31988509b)


**Step 3: Download question**

Go to Coursera quiz -> Click **_Download_** button 

![Screenshot from 2023-06-02 03-04-22](https://github.com/Joycean0301/Courseraaa/assets/103662477/ea7e1c09-156b-485b-ae00-14366c87a125)


**Step 4: Merge txt to csv file**

Create `txt_folder` in folder git -> Put all txt in -> Run file `merge_txt.py` -> output: `file.csv`

![Screenshot from 2023-06-02 03-05-09](https://github.com/Joycean0301/Courseraaa/assets/103662477/f741b021-2efb-4856-b7da-d83d14a092c3)
